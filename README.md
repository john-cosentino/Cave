# Personal Linux Lab

This repository is for personal Linux, Git, scripting, Ansible, and systems administration practice.

## Main Areas

- `notes/` - Study notes and reference material
- `labs/` - Hands-on exercises and experiments
- `scripts/` - Bash and Python scripts
- `ansible/` - Playbooks, roles, inventories, and Ansible notes
- `docs/` - Longer documentation
- `scratch/` - Temporary experiments

## Rules

- No work-owned code
- No company hostnames, IPs, credentials, logs, or internal documentation
- No secrets or tokens
- Keep examples generic and sanitized
