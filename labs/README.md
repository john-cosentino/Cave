# Labs

This directory contains hands-on practice for Linux, scripting, Ansible, RHEL-style administration, and containers.

## Lab Areas

- `bash/` - Bash and shell scripting practice
- `python/` - Python scripting practice
- `ansible/` - Ansible playbooks and inventories
- `rhel/` - Red Hat-style administration notes and labs
- `containers/` - Container experiments

## Lab Format

Each lab should include:

1. Objective
2. Commands used
3. What happened
4. What I learned
5. Follow-up questions

## Personal Rule

Every lab should teach something specific and leave behind notes that future me can understand.
