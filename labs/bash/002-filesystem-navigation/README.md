## What I Learned

- `tree` gives a better visual overview of directories than `ls`.
- `find` is useful when searching through nested directories.
- `du` helps identify disk usage.
- `stat` shows detailed file metadata, including permissions, ownership, timestamps, and inode information.
- `head` and `tail` are safer than dumping an entire file when I only need the beginning or end.
